[ผู้ดูแลระบบ IT Support | ผู้ช่วยนักพัฒนาซอฟต์แวร์]

ผู้เชี่ยวชาญด้าน IT Support ที่มีประสบการณ์ในการแก้ไขปัญหาด้านเทคนิค, ดูแลระบบ และมีความสนใจในการพัฒนาซอฟต์แวร์สำหรับ Web และ Windows Application ในระดับพื้นฐาน

ติดต่อฉัน ดูผลงาน

เกี่ยวกับฉัน

ผมเป็นผู้ดูแลระบบ IT Support ที่มีประสบการณ์ [จำนวน] ปีในการให้ความช่วยเหลือทางเทคนิค, การบำรุงรักษาระบบคอมพิวเตอร์และเครือข่าย, รวมถึงการติดตั้งและแก้ไขปัญหาซอฟต์แวร์ นอกจากนี้ ผมยังมีความสนใจและทักษะพื้นฐานในการพัฒนาซอฟต์แวร์ประเภท Web Application และ Windows Application เพื่อเพิ่มประสิทธิภาพในการทำงาน และพร้อมเรียนรู้สิ่งใหม่ๆ เพื่อเติบโตในสายงาน IT ครับ

ทักษะ

ทักษะด้าน IT Support

  • การแก้ไขปัญหา Hardware/Software 90%
  • การจัดการเครือข่ายพื้นฐาน 80%
  • ระบบปฏิบัติการ (Windows, Linux Basic) 85%
  • การติดตั้งและบำรุงรักษาซอฟต์แวร์ 95%

ทักษะการพัฒนาซอฟต์แวร์ (พื้นฐาน)

  • HTML, CSS (Web App Basic) 70%
  • JavaScript (Basic) 60%
  • Python/C# (Windows App Basic) 50%
  • Database (SQL Basic) 75%

ประสบการณ์ทำงาน

IT Support Specialist

บริษัท เทคโนโลยีรุ่งเรือง จำกัด | กรุงเทพฯ, ประเทศไทย | ม.ค. 2564 - ปัจจุบัน

  • ให้การสนับสนุนทางเทคนิคแก่ผู้ใช้งานทั้ง Hardware และ Software ภายในองค์กรมากกว่า 100 คน
  • แก้ไขปัญหาเครือข่ายเบื้องต้น, การเชื่อมต่อ Wi-Fi และ VPN เพื่อให้การทำงานราบรื่น
  • ติดตั้ง, กำหนดค่า และบำรุงรักษาเครื่องคอมพิวเตอร์และอุปกรณ์ต่อพ่วงต่างๆ
  • พัฒนาระบบ Internal Web Application ขนาดเล็กด้วย HTML/CSS/JS เพื่อช่วยในการจัดการคำร้องขอ IT

Junior IT Support

บริษัท โซลูชั่นไอที จำกัด | เชียงใหม่, ประเทศไทย | ก.พ. 2562 - ธ.ค. 2563

  • รับแจ้งปัญหาและให้คำปรึกษาด้านเทคนิคแก่พนักงานทางโทรศัพท์และอีเมล
  • ดูแลและบำรุงรักษาระบบคอมพิวเตอร์และซอฟต์แวร์พื้นฐาน
  • ช่วยเหลือในการจัดทำเอกสารคู่มือการใช้งานสำหรับซอฟต์แวร์ภายใน
  • เรียนรู้การสร้าง Windows Application พื้นฐานเพื่อจัดการ Asset Inventory ของอุปกรณ์ IT
